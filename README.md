## Home Library
----------------------------

The application provides you with a home library with some useful features (for example, a rating system).
The application uses the following technologies:
- Java 11.0.8;
- Project Lombok;
- Maven;
- PostgreSQL;
- Docker;
- Swagger;
- SLF4J;
- Junit 5 Upiter;
- Mockito;
- Spring Framework:
  - Spring Boot
  - Spring Data JPA
  - Spring WEB
  - Spring Security
  - Spring Test
